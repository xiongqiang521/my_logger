


# 任务进度
* 前端，查看日志的展示

## 已实现
* 在方法上添加注解，打印日志
* 将每个方法的进出打印日志
* 在类上添加注解可以实现每个方法都实现日志
* spring的@Controller、@Service、@Component自动打印日志


## 目标
* 构建springboot-start
* 添加调用链，确认每个请求所走过的方法
* 可选择是否存入数据库中
* 将每个方法的进出打印日志
* 静态方法的进出方法栈打印日志
* 方法体中间可以添加日志，例如：logger.info("日志信息");
* 按用户添加查看的权限

## 问题及优化
* 数据库设计是否合理
* 能否扩展
* 代码规范性
